# Практика 5 — REST API + Swagger

Доработка практики 4: сервер с API пользователей и документацией Swagger.

**Сервер:** Express, CRUD `/api/users`, Swagger UI по адресу `/api-docs` (JSDoc-аннотации).

**Клиент:** React (Vite), страница пользователей с запросами к API.

**Запуск**

Сервер:
```
cd practice_5/server
npm install
npm start
```
Swagger: http://localhost:3000/api-docs

Клиент (второй терминал):
```
cd practice_5/React_app_practice_5
npm install
npm start
```
Сайт: http://localhost:3001
